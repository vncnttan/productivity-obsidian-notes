### Variable
```python
name = "NJ"
words = "hello"

print("%s said %s"%(name, words))
print('{} said {}'.format(name, words))
print('{1} kata {0}'.format(name, words))
print(f'{name} said {words}')

phi = 3.1428
print("{:2f}".format(phi))
print("%.2f"%(phi))
print(f"{phi:.2f}")
```
