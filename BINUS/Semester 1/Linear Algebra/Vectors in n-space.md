**v** = (v1, v2, ..., vn)  `Vector in n-space`

The tuple set of (v1, v2..., vn) is called `n-space`, denoted by R<sup>n</sup>

**0** = (0, 0, ..., vn) is called the `zero vector`

### Addition
**v + w** = (v1 + w1, v2 + w2, .... vn + wn)

### Substraction
**v - w** = (v1 - w1, v2 - w2, .... vn - wn)

### Scalar Multiplication
n**v** = (v1 * n, v2 * n, .... vn * n)
